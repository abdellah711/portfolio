A developer experience (DX) tool designed to streamline the Shadcn UI theming workflow. Solves the complexity of managing raw HSL CSS variables by providing a real-time visual editor. Users can instantly generate and preview accessible color palettes, exporting compliant CSS code directly to their IDE
